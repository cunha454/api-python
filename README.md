# api-python
Projeto desenvolvido para praticar o consumo de APIs utilizando Python e a biblioteca requests, implementando operações básicas de CRUD.

api_biblioteca_autor.py: permite cadastar, listar, editar e apagar autores, contendo nome, nacionalidade e data de nascimento.

api_trabalho_servico.py: permite cadastrar, listar, editar e apagar serviços, contendo nome, preço e duração. Nessa comunicação com a API foi implementado a biblioteca Rich, melhorando a interface do terminal.

api_biblioteca.py: api_biblioteca.py: permite cadastrar, listar, editar e apagar bibliotecas, contendo nome, endereço, cidade, estado, CEP, telefone, e-mail, website, horário de atendimento e data de criação. Nessa comunicação com a API foi implementada a biblioteca Rich e a biblioteca Questionary, proporcionando uma interface mais organizada e prática no terminal.
